# 第二影音資料庫

這個專案提供一組保守使用的本地媒體擷取流程，讓後續 agent 可以做轉錄、摘要、腳本分析、剪輯規劃、分類與知識庫整理。

## 功能

- `scripts/download-video.ps1`：輸入網址，下載影片到 `outputs/media/videos/`
- `scripts/extract-audio.ps1`：只抓音訊，輸出到 `outputs/media/audio/`，方便後續轉錄
- `scripts/extract-subtitles.ps1`：只抓字幕，不下載影片，輸出到 `outputs/media/subtitles/`
- `scripts/metadata-only.ps1`：只輸出 metadata JSON，輸出到 `outputs/media/metadata/`
- `batch-list.txt`：一次處理多個網址，每行一個 URL，空白行和 `#` 註解會被忽略
- `skills/media-ingestion/`：給 Codex/agent 使用的專案內技能說明

## 快速使用

```powershell
.\scripts\download-video.ps1 -Url "https://example.com/video"
.\scripts\extract-audio.ps1 -Url "https://example.com/video"
.\scripts\extract-subtitles.ps1 -Url "https://example.com/video" -IncludeAutoGenerated
.\scripts\metadata-only.ps1 -Url "https://example.com/video"
```

批次處理：

```powershell
.\scripts\metadata-only.ps1 -BatchFile .\batch-list.txt
.\scripts\extract-subtitles.ps1 -BatchFile .\batch-list.txt -IncludeAutoGenerated
.\scripts\extract-audio.ps1 -BatchFile .\batch-list.txt
```

檢查命令但不下載：

```powershell
.\scripts\download-video.ps1 -Url "https://example.com/video" -DryRun
```

預設會加上 `--no-playlist`，避免不小心抓完整播放清單。確定要處理 playlist 時才加：

```powershell
.\scripts\metadata-only.ps1 -Url "https://example.com/playlist" -AllowPlaylist
```


## 建議流程

1. 授權或公開可用影片：先跑 `metadata-only.ps1` 和 `extract-subtitles.ps1`。
2. 沒有字幕但可合法處理：跑 `extract-audio.ps1`，再交給轉錄 agent。
3. 需要本地參考影片時：才跑 `download-video.ps1`。
4. 後續 agent 使用字幕、音訊、metadata 做摘要、分類、來源整理、剪輯規劃或知識庫匯入。
